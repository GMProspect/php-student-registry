## 📖 About the Project
A robust, vanilla PHP application designed to manage student enrollments, grades, and academic records seamlessly. This project emphasizes backend logic, relational database management, and raw performance without the overhead of heavy frameworks.

## ✨ Key Features
- ⚡ **Raw Performance**: Fast, server-side rendered interfaces powered by Vanilla PHP.\n- 🔐 **Role-Based Access**: Secure authentication and role-based access control for staff and admins.\n- 🗄️ **Optimized SQL**: Efficient SQL queries and strict database structuring.
